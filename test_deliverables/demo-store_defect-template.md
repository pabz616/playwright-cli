### Steps to Reproduce

Clear and succinct steps (do not use, "notice this" type of language)

1. Open the testing website {url}
2. Click for desktop applications || Tap for mobile applications
3. Scroll to ...
4. Type the following text ...
5. Click submit || Tap the submit button

### Expected Result

What is the intended outcome when carrying out the steps

### Actual Result

What is the observed action or behavior encountered when carrying out the steps

### Screenshot

Include a highlighted screenshot (e.g. red rectangle around the bug) in .jpg or .png format

### Videos

### Error Message

Write the full error message that is displayed on the screen if the bug results in an error message.

* Feature / Section
* Branch name
* Environment name
* Compare w. Production
